# Expense Tracker (Console-based)

A Java application that allows users to manage their finances through a secure login system. This project demonstrates core **Java 17** features and **Object-Oriented Programming (OOP)**.

### ✨ Key Features
- **User Authentication:** Secure registration and login.
- **Data Persistence:** Uses Serialization (`.ser`) to save user data.
- **Financial Management:** Add incomes, expenses, and recurring costs.
- **Budgeting:** Set monthly and yearly targets with a summary view.

### 🛠 Tech Stack
- **Language:** Java 17
- **Libraries:** `java.util`, `java.time`, `java.io` (Serialization)
